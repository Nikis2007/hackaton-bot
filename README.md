1. Рабочий телеграм-бот, который:

корректно принимает файлы истории чатов Telegram;
извлекает всех уникальных участников чата, которые писали сообщения;
упоминания пользователей в тексте сообщений извлекаются отдельно и не включаются в основной список участников;
не содержит дублей пользователей;
формирует результат в соответствии с условиями кейса (список в чате либо Excel-файл);
работает без сохранения пользовательских данных и файлов на сервере.
2. Технические артефакты по основным задачам:

репозиторий на GitHub с исходным кодом решения;
инструкция по запуску бота;
пример результата работы бота (Excel-файл и/или скриншоты).
3. Документация по основным задачам:

краткая инструкция для пользователя;
описание технических ограничений решения (формат истории, количество файлов, ограничения обработки).
